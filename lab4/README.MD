# Lab4

## Task

By-symbol processing - change о and О to 0, i and I to 1.
By-string processing - change x to *.

## How to use
- Compile ```make```
- Run ```./lab -help``` to see help
- Run ```./lab -i <input file> -o <output file> -m <mode>```

File *data* is example input file.
